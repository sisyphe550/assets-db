# FAMS 前端美术设计规范

> 本文档为 AI 可直接落地的视觉设计权威来源。  
> 技术实现见 `01-design.md`，组件规格见 `05-components.md`。

## 1. 品牌标识

### 1.1 系统名称

| 场景 | 文案 |
|---|---|
| 完整名称 | 高校固定资产管理系统 |
| 简称 | FAMS |
| 英文副标题（登录页） | Fixed Assets Management System |

### 1.2 Logo

| 资源 | 路径 | 尺寸 | 用途 |
|---|---|---|---|
| 主 Logo | `public/logo.png` | 32×32 / 48×48 | 顶栏、登录页、Favicon |
| Favicon | `public/favicon.ico` | 16×16 | 浏览器标签 |

**顶栏 Logo 组合**：

```
[logo 32px]  高校固定资产管理系统
             ↑ 16px 白色粗体，与 logo 垂直居中
```

**登录页 Logo 组合**：

```
        [logo 64px]
   高校固定资产管理系统
 Fixed Assets Management System
        ↑ 20px 主色 #1677FF 粗体
        ↑ 12px #8c8c8c 常规
```

### 1.3 品牌气质

- **专业**：高校行政后台，信息密度高但不杂乱
- **可信**：深色导航 + 白卡片，强调数据准确性
- **高效**：操作路径短，状态一眼可辨

## 2. 设计令牌（Design Tokens）

### 2.1 色彩

```typescript
// styles/tokens.ts — AI 实现时创建此文件
export const colors = {
  // 品牌色
  primary: '#1677FF',
  primaryHover: '#4096FF',

  siderItemHover: '#ffffff14',      // 8% 白
  siderItemSelected: '#1677FF',

  // 语义色（对齐 Ant Design preset）
  success: '#52c41a',
  warning: '#faad14',
  error: '#ff4d4f',
  info: '#1677ff',
  processing: '#1677ff',

  // 文字色

  textSecondary: '#00000073',       // 45% 黑

```

### 2.2 字体

| 层级 | 字号 | 字重 | 行高 | 用途 |
|---|---|---|---|---|
| H1 | 20px | 600 | 28px | 页面标题（PageHeader） |
| H2 | 16px | 600 | 24px | 卡片标题、区块标题 |
| Body | 14px | 400 | 22px | 正文、表格、表单 |
| Caption | 12px | 400 | 20px | 辅助说明、时间戳 |
| Stat | 32px | 600 | 40px | 仪表盘统计数字 |
| Mono | 13px | 400 | 20px | 资产编号、工单号 |

```css
/* 全局字体栈 */
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto,
             'Helvetica Neue', Arial, 'Noto Sans SC', sans-serif;

/* 等宽字体（资产编号） */
font-family: 'SF Mono', Monaco, 'Cascadia Code', 'Consolas', monospace;
```

### 2.3 间距

| Token | 值 | 用途 |
|---|---|---|
| page-padding | 24px | 内容区内边距 |
| card-gap | 16px | 卡片/栅格间距 |
| section-gap | 24px | 页面区块间距 |
| form-item-gap | 24px | 表单项垂直间距 |
| button-gap | 8px | 按钮组间距 |
| inline-gap | 4px | 图标与文字间距 |

### 2.4 圆角与阴影

| Token | 值 | 用途 |
|---|---|---|
| radius-sm | 4px | Tag、小按钮 |
| radius-md | 6px | 输入框、卡片 |
| radius-lg | 8px | 登录卡片、大面板 |
| shadow-card | `0 1px 2px 0 rgba(0,0,0,0.03), 0 1px 6px -1px rgba(0,0,0,0.02)` | Card 组件 |
| shadow-drawer | `0 6px 16px 0 rgba(0,0,0,0.08)` | Drawer |

### 2.5 布局尺寸

| 元素 | 尺寸 |
|---|---|
| 顶栏高度 | 48px |
| 侧边栏展开宽度 | 220px |
| 侧边栏折叠宽度 | 80px |
| 内容区最小宽度 | 1200px（低于此显示横向滚动提示） |
| Drawer 宽度 | 640px |
| Modal 宽度 | 520px（默认）/ 720px（用户创建） |
| 登录卡片宽度 | 400px |

## 7. 图表视觉规范

使用 **@ant-design/charts**（基于 G2）。

| 图表 | 类型 | 配色 | 高度 |
|---|---|---|---|
| 按类别分布 | 饼图（Pie） | `['#1677FF','#52c41a','#faad14','#ff4d4f','#722ed1']` | 300px |
| 按部门统计 | 柱状图（Column） | 主色 `#1677FF` | 300px |
| 审批趋势 | 折线图（Line） | `#1677FF`，area 填充 10% 透明度 | 300px |

**通用配置**：

```typescript
const chartTheme = {
  animation: { appear: { animation: 'fade-in', duration: 500 } },
  legend: { position: 'bottom' },
  label: { style: { fontSize: 12 } },
};
```

## 8. 空状态 / 加载 / 错误视觉

### 8.1 空状态（Empty）

| 场景 | 插图 | 描述文案 | 操作 |
|---|---|---|---|
| 资产列表为空 | Ant Design 默认 | 暂无资产数据 | [新增资产]（role≤2） |
| 待审批为空 | 默认 | 暂无待审批工单 | — |
| 我的工单为空 | 默认 | 暂无申请记录 | [新建申请] |
| 盘点任务为空 | 默认 | 暂无盘点任务 | [创建任务]（role≤2） |
| 搜索无结果 | 默认 | 未找到匹配结果 | [清除筛选] |

### 8.2 加载态

| 场景 | 组件 | 说明 |
|---|---|---|
| 整页加载 | `<Spin size="large" tip="加载中...">` 居中 | 首次进入页面 |
| 表格加载 | ProTable 内置 skeleton | 翻页/筛选 |
| 按钮提交 | `loading={true}` | 禁用重复点击 |
| Univer 加载 | `<Skeleton active paragraph={{ rows: 10 }}>` | 懒加载组件 |
| 导出进度 | `<Progress percent={n} status="active">` | Modal 内 |

### 8.3 错误态

| 场景 | 组件 | 配色 |
|---|---|---|
| 页面加载失败 | `<Result status="error">` + 重试按钮 | — |
| 资源不存在 | `<Result status="404">` | — |
| 权限不足 | `<Result status="403" title="无权访问">` | — |
| 表单字段错误 | 红色边框 + 字段下方红字 | `#ff4d4f` |
| 盘点冲突行 | 行背景 `#fff2f0` | 红色左边框 3px `#ff4d4f` |
| 盘点成功行 | 行背景 `#f6ffed` | 绿色左边框 3px `#52c41a` |

## 11. 无障碍与国际化

### 11.1 v1 范围

- 语言：仅中文（zh-CN）
- Ant Design `ConfigProvider locale={zhCN}`
- Univer `LocaleType.ZH_CN`
- 日期格式：`YYYY-MM-DD`（展示）/ ISO8601（API 传输）

### 11.2 无障碍最低要求

- 所有图标按钮添加 `aria-label`
- 表单字段关联 `<label>`
- 颜色不作为唯一信息载体（状态标签同时显示文字）
